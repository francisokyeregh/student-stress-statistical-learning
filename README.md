# An Interpretable Statistical Learning Framework for Binary Classification: An Application to Student Stress Prediction

This repository contains the R code, figures, tables, and supplementary materials for the manuscript:

> **An Interpretable Statistical Learning Framework for Binary Classification: An Application to Student Stress Prediction**

## Authors

Francis Okyere
PhD Student

Department of Statistics  
Florida State University  
Tallahassee, FL, USA

---

## Abstract

This repository accompanies the manuscript submitted for publication. It provides the complete computational workflow used for data preprocessing, statistical modeling, bootstrap stability selection, explainable machine learning, and generation of all figures and tables.

---

## Dataset

The data used in this study are publicly available from Kaggle:

https://www.kaggle.com/datasets/sridevilavanyacse/student-lifestyle-and-stress-prediction-dataset
